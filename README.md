# 🎮 Toryloria

Toryloria is a browser-based game platform where users can create, share, and play 2D games within a stylized Bento Grid aesthetic.

## 🚀 Deployment Instructions

### 1. GitHub
To upload this to GitHub:
1. Initialize a new repo: `git init`
2. Add files: `git add .`
3. Commit: `git commit -m "Initial commit"`
4. Link to GitHub: `git remote add origin YOUR_REPOSITORY_URL`
5. Push: `git push -u origin main`

*Tip: You can use the "Export to GitHub" option in the AI Studio Settings menu.*

### 2. Vercel
Once on GitHub:
1. Go to [Vercel](https://vercel.com)
2. Import your GitHub repository.
3. Vercel will automatically detect the Vite setup.
4. Click **Deploy**.

## 🛠 Features
- **Identity Portal**: Local-stored account system (Login/Register).
- **World Studio**: 2D Map editor with grid-based placement.
- **Game Engine**: Physics-based 2D gameplay with win/loss states.
- **Avatar Hub**: Deep character customization and clothing system.
- **Dual Protocols**: Optimized PC (Keyboard) and Mobile (Touch) controls.

## 📦 Tech Stack
- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
